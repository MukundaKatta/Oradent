# Oradent

> Dental Practice Management Platform

## Overview

Oradent is a comprehensive dental practice management system for appointments, patient records, treatment planning, and billing. Designed for dental clinics of all sizes.

## Key Features

- **Appointment Scheduling** — Calendar management with automated reminders
- **Patient Records** — Complete dental history, X-rays, and treatment notes
- **Treatment Planning** — Visual dental charting and treatment plan creation
- **Billing & Insurance** — Invoice generation, insurance claims, payment tracking
- **Reporting** — Practice analytics, revenue reports, patient demographics
- **Multi-Provider** — Support for multiple dentists and hygienists

## Tech Stack

React, Node.js, Firebase, Material-UI

## Getting Started

```bash
git clone https://github.com/MukundaKatta/Oradent.git
cd Oradent && npm install && npm start
```
